O que é HTML?

HTML, ou Linguagem de Marcação de Hipertexto, é a linguagem de programação padrão para criar e estruturar páginas web, sendo o "esqueleto" que define o conteúdo e a organização de um site. Ela utiliza uma série de "tags" para marcar textos, imagens, links e outros elementos, permitindo que os navegadores exibam o conteúdo de forma estruturada e significativa.
